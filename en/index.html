PixelARC

PixelARC is a programming language conceived to allow the generation of images in the "pixel art" style.

The project is composed of two major parts:

  • The compiler (converts the PixelARC code into bytecode)
  • The interpreter (interprets the bytecode and generates the final image)

The language implements some concepts such as variables, conditions, loops, functions, etc.
Native functions are also implemented, such as trigonometric functions and functions that allow you to draw on the final image.

Arquizz

Arquizz is a website inspired by Kahoot! that allows you to create and play quizzes online in real-time against other users. The project implements the following features:

  • User management (registration, login, profile)
  • Quiz management (creation, modification, deletion)
  • Question and answer management
  • Public room system (game that can be joined by anyone using a code)
  • Points and ranking system for a game
  • Time system for a game (time limit per question)
  • etc.

Technologies used: HTML, CSS, JavaScript, PHP, Laravel, MySQL

Grafile

Grafile is a software conceived to automate some actions on files, such as renaming, moving, deleting, etc. The interface is in the form of a node system that can be linked together.

Technologies used: C++, Qt
