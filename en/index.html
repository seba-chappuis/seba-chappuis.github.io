Bachelor project

The bachelor's project, completed individually, consists of two distinct parts:

  • Utility for generating mazes
  • 3D game for navigating generated mazes

The utility, inspired by the Labyrinth Path Solver project described above, allows you to generate and visualize mazes composed of isolated areas called clusters (each cluster is represented by a distinct color).

The clusters are then connected by teleporters so that each cluster is accessible (each pair of teleporters is represented by a letter).

A maze is composed of multiple rooms, each with an associated difficulty (represented by a number from 0 to 100). The closer the room is to the finish line, the higher its difficulty.

The generated mazes can be saved as JSON files. An algorithm is used to determine the global difficulty level of a maze.

The 3D game allows players to explore a series of mazes of increasing difficulty.

Multiple rooms composed of obstacles must be navigated, the obstacles varying depending on the room's difficulty.

The player can view the maze map at any time, showing their current location and the position of the teleporters. However, the player does not know where the teleporters lead. They must therefore identify the room's difficulty to determine whether they are getting closer or further from the destination.

The player has three health points. If they fall into the void, they lose a life and reappear at the entrance to the current room. If they lose all their health points, it's game over; the number of mazes completed represents the player's score.

Technologies used for the utility: Python, Tkinter (graphical interface), JSON

Technologies used for the 3D game: Godot (game engine), Blender (3D modeling), JSON

PixelARC

PixelARC is a programming language conceived to allow the generation of images in the "pixel art" style.

The project is composed of two major parts:

  • The compiler (converts the PixelARC code into bytecode)
  • The interpreter (interprets the bytecode and generates the final image)

The language implements some concepts such as variables, conditions, loops, functions, etc.
Native functions are also implemented, such as trigonometric functions and functions that allow you to draw on the final image.

Technologies used: Python, Numpy, Pillow, Ply, PyDot

Arquizz

Arquizz is a website inspired by Kahoot! that allows you to create and play quizzes online in real-time against other users. The project implements the following features:

  • User management (registration, login, profile)
  • Quiz management (creation, modification, deletion)
  • Question and answer management
  • Public room system (game that can be joined by anyone using a code)
  • Points and ranking system for a game
  • Time system for a game (time limit per question)
  • etc.

Technologies used: HTML, CSS, JavaScript, PHP, Laravel, MySQL
